# Мониторинг системы

## Общее состояние

```bash
uptime                         # нагрузка за 1/5/15 минут
top                            # интерактивный мониторинг процессов
htop                           # удобный интерфейс (если установлен)
watch -n5 'uptime && free -h'  # повторять команды каждые 5 секунд
```

## Память и swap

```bash
free -h                        # сводка по памяти
vmstat 5                       # страницы памяти и swap каждые 5 сек
smem -kt                       # топ потребителей памяти (если установлен smem)
```

## CPU и диски

```bash
mpstat 1                       # загрузка CPU по ядрам
iostat -xz 1                   # задержки диска и throughput
sar -u 1 5                     # статистика CPU (пакет sysstat)
```

## Файловые системы

```bash
df -h                          # свободное место по разделам
du -sh *                       # размер каталогов в текущем каталоге
ncdu                           # интерактивный анализ (если установлен)
```
