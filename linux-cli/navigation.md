# Навигация по файловой системе

## Базовые команды

```bash
pwd                # показать текущий каталог
ls -lha            # детальный список с правами, размерами и скрытыми файлами
ls -t              # сортировка по времени изменения
tree -L 2          # дерево каталогов с ограничением глубины
```

## Быстрые переходы

```bash
cd ~               # домой
cd -               # назад в предыдущий каталог
pushd /path && ... && popd  # временно перейти и вернуться
```

```bash
alias ..='cd ..'
alias ...='cd ../..'
```

## Работа со скрытыми файлами и ссылки

```bash
ls -A              # показать скрытые файлы (имена начинаются с .)
ls -ld .*          # права и владельцы скрытых каталогов
readlink -f file   # разыменовать символическую ссылку
```

## Быстрый поиск каталога

```bash
fd --type d pattern ~/projects   # быстрый поиск каталога по имени
autojump enable && j docs        # прыжок в каталог по истории (если установлен autojump)
```
